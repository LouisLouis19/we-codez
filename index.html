Class Concepts

Variables

What is a Variable?

Variable is a place where you can store information (like numbers, words, or other data) and give it a name so you can easily use or modify it later.

Example: You have a variable called 'myAge', which can be used to store you actual age.

This is how you create (declare) your variable.

myAge = 21

This is how you can use your variable, for example to print its value to your screen.

print(myAge)

This is how you can change (modify) your variable, for example to assign a new value to it.

age = 22
print(myAge) # myAge variable will now show 22

Arithmetic Operations

What is an Arithmetic Operation in Python?

It is basic calculations we perform using numbers, in Python it is similar to what you do with a calculator.

Here are some arithmetic operation types supported in Python:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Modulus

Addition Example:

result = 4 + 2
print(result)

Subtraction Example:

result = 10 - 2
print(result)

Multiplication Example:

result = 4 * 3
print(result)

Division Example:

result = 7 / 2
print(result)

Modulus Example:

result = 5 % 2
print(result)

Comparison Operations

What is a Comparison Operation in Python?

It is used to compare two values and see if they are equal, not equal, greater than, or less than each other. In Python, these operations return either True or False

Here are some comparison operation types supported in Python:

  • Equal To
  • Not Equal To
  • Greater Than
  • Less Than
  • Greater Than or Equal To
  • Less Than or Equal To

Equal To Example:

result = (5 == 5)
print(result)

Not Equal To Example:

result = (5 != 5)
print(result)

Greater Than Example:

result = (7 > 5)
print(result)

Less Than Example:

result = (5 < 6)
print(result)

Greater Than or Equal To Example:

result = (5 >= 5)
print(result)

Less Than or Equal To Example:

result = (7 <= 5)
print(result)

Control Flow - Conditional Statements

Control flow is like the decision-making part of your code.

It lets your program make choices based on logic (conditions) you define

Here are some conditional statements supported in Python:

  • if
  • elif
  • else

"if" Example:

temperature = 40
if temparature > 35:
print("It is hot outside.")

"elif" Example:

temperature = 30
if temperature > 35:
print("It is hot outside.")
elif temperature > 20:
print("It is warm outside.")

"else" Example:

temperature = 30
if temperature > 35:
print("It is hot outside.")
elif temperature > 20:
print("It is warm outside.")
else:
print("It is cold outside")
